# Entrega_Atividade_IV_SQL

# Projeto SQL – Sistema FitPower

Este repositório contém scripts SQL completos de criação, manipulação e consulta do banco de dados FitPower.

## Estrutura
- /scripts → SQL de criação, inserts, selects, updates, deletes
- /model → projeto lógico, DER, documentação

## Execução
1. Crie o schema:
   ```
   CREATE DATABASE fitpower;
   USE fitpower;
   ```
2. Execute create_tables.sql
3. Execute insert_data.sql
4. Utilize os demais scripts conforme necessário.

## Tecnologias
- MySQL Workbench ou PGAdmin
- SQL
- Git / GitHub
